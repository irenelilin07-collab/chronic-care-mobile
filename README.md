# 慢病用药小管家（移动端）

React + Vite + Tailwind 移动端 Web App，帮助慢性病患者管理用药、药箱库存与复诊计划。

## 已实现功能

### 用药计划
- 日 / 周 / 月看板与打卡进度
- 打卡联动药箱库存扣减
- 右上角 + 新建与管理用药计划

### 我的药箱
- 药品库存与按计划估算可用天数
- 卡片编辑 / 删除，右上角 + 添加

### 复诊计划
- 复诊倒计时、过期提醒、线上续方链接
- 右上角 + 添加复诊计划

### 设置
- 用药提醒（每个任务可提前 5 / 10 / 15 / 30 分钟浏览器通知）
- 导出用药报告（可选时段，导出 PDF）
- 基本信息、确诊慢病、药物过敏史、紧急联系人

## 本地运行

```bash
cd "/Users/bytedance/Desktop/慢性病管家"
npm install
npm run dev
```

浏览器打开终端显示的地址（通常是 `http://localhost:5173`）。

## 数据存储

所有数据保存在浏览器 LocalStorage（键名 `chronic-care-mobile-v1`）。可在「设置 → 导出用药报告」选择时段并下载 PDF 报告。
